# Gulpplate

Gulpplate is a free starter template built with Gulp and TailwindCSS, providing everything you need to jumpstart your Gulp project and save valuable time.

## Dependency

- node v18+
- npm v9.5+
- gulp v4+

## Package Installation

```
npm install
```

## Development Command

```
npm run dev
```

## Build Command

```
npm run build
```
